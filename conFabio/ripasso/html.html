HEAD

Nella prima parte del file html, si inseriscono i seguenti campi:

  • meta: contiene informazioni sulla tastiera utilizzata, la lingua e altre info sulla compatibilità con i diversi browser
  • title: rappresenta la stringa che comparirà nella Tab
  • link: inserisco il path del file css
  • NB: in presenza di altri link css, inserire per ultimo il collegamento la foglio
  • script: inserisco il path del file js
  • NB: inserire 'defer' o 'async' per risolvere i problemi si caricamento pagina wrt javascript

BODY

PRINCIPALI TAG
  • hNumero -> Titolo (< hNumero > è inversamente proporzionale alla sua grandezza)
  • p -> paragrafo classico
  • ul -> per creare un 'unorderd list'
  • li -> elementi che compongono la lista
  • br (inline)
    • -> lo utilizzo per andare a capo all'interno di un elemento
  • span (inline)
    • -> lo utilizzo per selezionare un elemento all'interno di un altro elemento
  • span (inline)
    • -> lo utilizzo per selezionare un elemento all'interno di un altro elemento
  • strong (inline) -> per evidenziare porzioni di testo (x dare piu importanza tipo google)
  • img (inline) -> per inserire un immagine
  • NB: si apre e chiude su stessa
  • Richiede gli attributi alt='stringa se non compare img', src='link/percorso dell'img src="" alt=""
  • inserire storia padding
  • input -> inserisce una casella (che puo essere riempita o usata per un bottone
  • NB: si apre e chiude su stessa e richiede gli attributi alt='stringa se non compare img', src='link/percorso dell'img src="" alt=""
  • Richiede gli attributi type='tipo input', value='scrittina nella casella', inoltre viene usato da collegamento nel back end
